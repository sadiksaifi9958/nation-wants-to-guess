# Nation Wants To Guess

A quiz web app based on the comedy show by Gursimran Khamba.

**Live demo:** [nwtg-quiz.vercel.app](https://nwtg-quiz.vercel.app)

![Nation Wants To Guess](public/screenshot.png)

## Features
- Random questions fetched from a live API
- 30 second countdown timer per question
- One wrong answer ends the game
- High score tracking
- Error handling with retry option
- Fully responsive

## Tech Stack
- React
- Vite
- Tailwind CSS
- Framer Motion

## Run locally
```
git clone https://github.com/sadiksaifi9958/nation-wants-to-guess.git
cd nation-wants-to-guess
npm install
npm run dev
```
